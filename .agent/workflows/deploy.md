// turbo-all

1. Create a repository on GitHub named "cargo-trim-app".
2. Initialize local git: `git init`
3. Add files: `git add .`
4. Commit: `git commit -m "First release v1.7.6"`
5. Rename branch: `git branch -M main`
6. Add remote: `git remote add origin https://github.com/USERNAME/cargo-trim-app.git` (YOU MUST REPLACE USERNAME)
7. Push: `git push -u origin main`

To share:

1. Go to Repository Settings > Pages.
2. Select "main" branch and save.
3. Share the URL provided by GitHub.
